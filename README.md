# dawaShare

A rideshare platform for Stanford students to find carpools to airports.

## About

dawaShare helps Stanford students coordinate rides to and from airports by:
- Submitting flight information through a simple form
- Viewing other students' travel plans
- Finding matches for carpooling opportunities
- Maintaining privacy while facilitating connections

## Features (Coming Soon)

- Flight info submission form
- Airport and time matching
- Email-based contact system
- Privacy controls for contact information
- Auto-filtered terminal matching

## Tech Stack

- HTML/CSS/JavaScript
- GitHub Pages for hosting
- Local CSV storage for ride data
